# Go Timer

A timer for board game players with voice alerts. Essential for Go enthusiasts.

## Documentation

- [Install](#install)
- [Release android](#release-android)
- [Features](#features)
- [Store](#store)
- [App icon generator](#app-icon-generator)

## Install

```shell
yarn install
npm run pod:install
```

## Release android

```shell
npm run cp:android
npm run release:android
```
